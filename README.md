# LTSOE
Learning to Score Olympic Events

This repository contains the following frameworks, which were used in our paper "Learning to Score Olympic Events":
1. C3D-SVR
2. C3D-LSTM
3. C3D-LSTM-SVR

We found and fixed some bugs in our codes. Please read "Learning_To_Score_Olympic_Events_Updated_Results.pdf" for all the updated results.
